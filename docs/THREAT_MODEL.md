# Threat Model

---

## Assets

- Track integrity
- Threat scoring accuracy
- Audit logs
- Operator trust

---

## Threats

- Unauthorized ingestion
- Data spoofing
- Privilege escalation
- State corruption

---

## Mitigations

- JWT authentication
- Schema validation
- Deterministic IDs
- Stateless recovery

---

## Out of Scope

- Nation-state cyber ops
- Physical compromise
- Classified data handling
